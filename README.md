# ❌⭕ Tic Tac Toe

🎭 *A simple game of strategy, where every move tells a story.*

A browser-based Tic Tac Toe game built using JavaScript, featuring a clean UI and interactive gameplay for two players.

---

## 🚀 Features

* 🎮 Two-player gameplay
* ⚡ Instant win/draw detection
* 🔄 Game reset functionality
* 💻 Responsive and minimal UI
* 🧩 Smooth user interaction

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Logic Handling:** Vanilla JavaScript
* **Version Control:** Git & GitHub

---

## ⚙️ How It Works

1. Players take turns marking ❌ and ⭕
2. Game checks for winning combinations after each move
3. Displays winner or draw result
4. Option to reset and play again

---

## 📦 Installation & Setup

Clone the repository:

```bash id="clone1"
git clone https://github.com/Lekhak666/Tic-Tac-Toe.git
```

Navigate into the folder:

```bash id="cd1"
cd Tic-Tac-Toe
```

Open `index.html` in your browser to play.
